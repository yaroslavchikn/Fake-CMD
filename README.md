# Fake CMD - Полная копия командной строки

Это приложение выглядит и работает как настоящая командная строка Windows, но на самом деле ничего не выполняет!

## Особенности

- Полная эмуляция командной строки Windows
- Поддержка популярных команд: dir, ipconfig, ping, echo, cls, help, exit
- Реалистичный вывод команд
- Поддержка аргументов (например, ping google.com)
- Сохранение истории команд

## Доступные команды

- `dir` - Показать содержимое папки
- `ipconfig` - Показать сетевые настройки
- `ping` - Проверить сетевое соединение
- `echo` - Вывести текст
- `cls` - Очистить экран
- `help` - Показать справку
- `exit` - Выйти из программы
- `time` - Показать текущее время
- `date` - Показать текущую дату
- `ver` - Показать версию ОС
- `systeminfo` - Показать информацию о системе
- `hostname` - Показать имя компьютера
- `whoami` - Показать имя пользователя
